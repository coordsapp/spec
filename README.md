# spec
Open protocol spec for Coords - CC0 public domain

## Versions
- `v1/geometry.md`: L1 URI structure and canonical encoding
- `v1/checksum.md`: v1 checksum algorithm
- `v1/altitude.md`: altitude unit and precision rules
- `v1/boundaries.md`: valid numeric ranges
- `v1/test-vectors.md`: reference vectors for encode/decode verification
- `v1/test-vectors.json`: machine-readable test vectors
